Accessing Bash Shell on different operating systems


On Mac OS
=========

Search (command + spacebar) for "Terminal". Open it up and you are good to go.




On Windows 10
=============

1.	Go to the start menu and search for PowerShell
2.	Select "run as administrator"
3.	Enter the following command into power shell:

Enable-WindowsOptionalFeature -Online -FeatureName Microsoft-Windows-Subsystem-Linux

4.	When asked to confirm, type the letter Y and press enter.
5.	Restart your computer
6.	Go to the Windows store and search for "Linux"
7.	Select "run Linux on Windows"
8.	Select "Ubuntu"
9.	Click "install"
10.	Go to the start menu and search for "Ubuntu" and run

You should now be able to use the Bash commands outlined in this course
